# AI Travel Planner Android App

An AI-powered Android application that generates travel itineraries using **Google Gemini AI**.

Users can enter a destination and the app will automatically generate a **3-day travel itinerary with places to visit and descriptions**.

Built using **Kotlin**, **Jetpack Compose**, and **MVVM architecture**.

## Features

• AI-generated travel itineraries  
• Modern UI built with Jetpack Compose  
• MVVM architecture  
• Google Gemini AI integration  
• Scrollable travel plan results  
• Fast AI responses using Gemini Flash model  

---

## Tech Stack

- **Kotlin**
- **Android Studio**
- **Jetpack Compose**
- **Google Gemini API**
- **MVVM Architecture**

---

## How the App Works

1. User enters a travel destination.
2. The app sends a prompt to the **Google Gemini AI model**.
3. Gemini generates a **3-day travel itinerary**.
4. The itinerary is displayed in the application UI.
